# Shonen Jump Grid

给任天堂NDS平台粉丝向游戏《JUMP ULTIMATE STARS》的一份情书
再次重现阅读漫画的感动
这是一个漫画过场动画编辑器应用，用户可以创建和编辑每一页，在每页划分漫画格子，同时在格子里上传图片并调整裁剪，最后可以通过预览按钮来看到自己把每个格子，每一页组合起来的效果。

## Setup

Install the dependencies:

```bash
pnpm install
```

## Get started

Start the dev server:

```bash
pnpm dev
```

Build the app for production:

```bash
pnpm build
```

Preview the production build locally:

```bash
pnpm preview
```
