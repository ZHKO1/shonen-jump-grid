# Shonen Jump Grid

给任天堂NDS平台粉丝向游戏《JUMP ULTIMATE STARS》的一份情书
再次重现阅读漫画的感动

## Setup

Install the dependencies:

```bash
pnpm install
```

## Get started

Start the dev server:

```bash
pnpm dev
```

Build the app for production:

```bash
pnpm build
```

Preview the production build locally:

```bash
pnpm preview
```
